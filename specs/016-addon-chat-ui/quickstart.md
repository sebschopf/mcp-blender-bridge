# Quickstart: Chat UI

## Usage

1.  **Open the Panel**: Press `N` in the 3D Viewport to open the sidebar.
2.  **Find Tab**: Look for the "Gemini MCP" tab.
3.  **Connect**: Click "Connect".
4.  **Chat**: 
    - Type in the text box at the bottom.
    - Press "Send".
    - See your message (User) and the response (AI) appear in the history above.

## UI Updates
- User messages have a 'USER' icon.
- AI messages have an 'ALIGN_LEFT' or distinct icon.
